BeFeed
============
Feed News Portal Indonesia

### How to run:
1. install `pip install -r requirements.txt`
2. run   
   `python app.py` or   
   `docker-compose up -d` or  
   `gunicorn app:app`
   

### How to run test:
1. install `pip install pytest`
2. run `pytest `
